# Фронтенд и бэкенд разработка — Продажа недвижимости

Практические работы по дисциплине «Фронтенд и бэкенд разработка», 4 семестр.  
Тема: **Продажа недвижимости**.

---

## Структура проекта

### Практика 1 — CSS-препроцессоры (`1-2/1/`)

Карточка объекта недвижимости, стилизованная с помощью **SASS**:

- Переменные (`$accent-color`, `$surface-color`, `$text-color`, `$radius`, `$shadow`)
- Миксин `cta-button` для кнопки
- Вложенная структура селекторов по методологии BEM

**Запуск:** открыть `index.html` в браузере.

### Практика 2 — Сервер на Node.js + Express (`1-2/2/`)

REST API с CRUD-операциями для объектов недвижимости:

| Метод    | Маршрут         | Описание                    |
|----------|-----------------|-----------------------------|
| GET      | `/products`     | Список объектов             |
| GET      | `/products/:id` | Объект по ID                |
| POST     | `/products`     | Создание объекта            |
| PATCH    | `/products/:id` | Обновление объекта          |
| DELETE   | `/products/:id` | Удаление объекта            |

**Запуск:**
```bash
cd 1-2/2
npm install
npm start
```

### Практика 3 — JSON и внешние API (`3-4/3/`)

Расширенный сервер с CRUD для недвижимости и прокси к ExchangeRate API:

| Метод | Маршрут                        | Описание                  |
|-------|--------------------------------|---------------------------|
| GET   | `/properties`                  | Список объектов           |
| GET   | `/properties/:id`              | Объект по ID              |
| POST  | `/properties`                  | Создание объекта          |
| PUT   | `/properties/:id`              | Полное обновление         |
| PATCH | `/properties/:id`              | Частичное обновление      |
| DELETE| `/properties/:id`              | Удаление объекта          |
| GET   | `/exchange-rate/:base`         | Курсы валют               |
| GET   | `/exchange-rate/:base?target=` | Курс конкретной валюты    |

Файл `requests.http` содержит примеры запросов для тестирования.

**Запуск:**
```bash
cd 3-4/3
npm install
npm start
```

### Практика 4 — API + React (`3-4/4/`)

Полноценное веб-приложение «Продажа недвижимости»:

- **Бэкенд** (Express, порт 3000): API с CRUD, CORS, валидацией, nanoid
- **Фронтенд** (React, порт 3001): список объектов, создание, редактирование, удаление

10 объектов недвижимости с полями: название, категория, описание, цена, количество.

**Запуск бэкенда:**
```bash
cd 3-4/4/backend
npm install
npm start
```

**Запуск фронтенда:**
```bash
cd 3-4/4/frontend
npm install
npm start
```

### Практика 5 — Swagger документация (`5-6/5/`)

Доработка приложения с интерактивной документацией Swagger:

- Бэкенд с полной JSDoc-аннотацией всех CRUD-эндпоинтов
- Схема `Property` описана через `@swagger components/schemas`
- Swagger UI доступен по адресу `http://localhost:3000/api-docs`
- Фронтенд с управлением объектами недвижимости

**Запуск бэкенда:**
```bash
cd 5-6/5/backend
npm install
npm start
```

**Запуск фронтенда:**
```bash
cd 5-6/5/frontend
npm install
npm start
```
